METALWORK JOBS - ONE UPLOAD PACKAGE

1. Extract this ZIP.
2. Open the extracted folder.
3. Select every file in the extracted folder, including the hidden .nojekyll file if your file manager shows it.
4. Upload all selected items to the repository root.
5. Do not rename index.html.
6. Remove old repository files before uploading this package.
7. After deployment, open the site in a private/incognito window for the first check.

This build keeps CSS and JavaScript inside each HTML page, while images remain normal root files. It contains no base64/data-URI images and no service worker cache.
